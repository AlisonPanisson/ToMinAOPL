# ToMinAOPL - ToM in AOPL

This project aims to investigate the modelling of Theory of Mind (ToM) in Agent-Oriented Programming Languages (AOPL).

Contact autors:

> * Alison R. Panisson -- alison.panisson@acad.pucrs.br
